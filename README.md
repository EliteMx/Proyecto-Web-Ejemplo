# Proyecto-Web-Ejemplo
Proyecto Web de Prueba

## Clonar y ejecutar el proyecto en un entorno local

1. Clona el repositorio:
   git clone https://github.com/EliteMx/Proyecto-Web-Ejemplo.git
2. Entrar a la Carpeta del Proyecto
   cd Proyecto-Web-Ejemplo
3. Abrir el Proyecto en VSCODE
   code .

## Ramas del proyecto

El proyecto contiene las siguientes ramas:

- **main**: La rama principal que contiene el código de producción.
- **feature-mejora-estilo**: Rama que implementa mejoras en el estilo de la página web.
